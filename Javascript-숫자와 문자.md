# JavaScript-숫자와 문자(2/4)
## 수의 연산

```javascript
Math.pow(2,3);
//2의 3제곱, 8
Math.round(13.7);
//13.7을 반올림, 14
Math.ceil(17.2);
//17.2를 올림, 18
Math.floor(2.5);
//2.5를 내림, 2
Math.sqrt(25);
//25의 제곱근, 5
Math.random();
//0부터 1.0 사이 임의의 숫자
```

# JavaScript-숫자와 문자(3/4)
## 문자의 표현

```javascript
alert("Hello World");
alert('Hello World');
alert(typeof "15");
//입력값의 데이터 형을 알려줌, string
alert(typeof 15);
//따옴표가 없으므로 number
alert('you\'re welcome');
//escape를 활용, 작은따옴표를 문자열 안에 포함시킴.
```
