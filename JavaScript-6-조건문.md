# 2021.10.30(SAT) JavaScript-조건문(1/5)
## 조건문이란
주어진 조건에 따라서 애플리케이션을 다륵 동작하도록 하는 것이다.

```javascript
if(true){
    alert(1);
    alert(2);
    alert(3);
    alert(4);
}
alert(5);   
```

```javascript
if(false){
    alert(1);
    alert(2);
    alert(3);
    alert(4);
}
alert(5);
```
