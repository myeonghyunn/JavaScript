# 2021.11.07 (SUN) JavaScript-함수(1/5)
## 함수란?

```html
<script type = "text/javascript">
function texting() {
document.write("Hello World");
}
texting();
</script>

```

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset = 'utf=8'>
    <title>Hello World</title>
  </head>
  <body>
    <script type = "text/javascript">
    function numbering() {
      i = 1;
      while(i<=10) {
        document.write("함수 반복 중입니다. 현재 "+i+"번째 반복입니다. <br />");
        i += 1;
      }
    }
      numbering();
    </script>
  </body>
</html>
```
