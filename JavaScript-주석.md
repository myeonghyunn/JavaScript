#  2021.10.30(SAT) JavaScript-주석

JavaScript 언어애서의 '//'는 특수한 의미를 갖는다.

'//' 뒤에 등장하는 내용들은 문법적으로 해석하지 않도록 명령한다.

정말 좋은 코드는 주석조차 달지않아도 이해할 수 있는 가독성이 좋은 코드라고 한다.
